Rules that make HTML document valid:

  • Always specify the document type as HTML at the top of the page
  • All work must be within the HTML tags just below the document type tag
  • Information in regards to the character set and style sheets is included in the head tag
  • The title that will show on the tab is included in the head tag
  • All information that the webpage will show needs to be included in the body tag
  • indent all children tags from parent tags
  • If you open a tag you must ensure you close the tag
